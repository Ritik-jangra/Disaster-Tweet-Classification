🧠 Pipeline

How It Works

Four deterministic steps transform raw tweet text into a calibrated prediction.

01
🧹

Text Cleaning

Strips URLs, @mentions, hashtags, punctuation, and stopwords. spaCy lemmatises tokens — running → run — to reduce vocabulary noise.

spaCy · Regex
02
📐

TF-IDF Vectorisation

15,000-feature TF-IDF with unigrams + bigrams converts cleaned text into a sparse numeric vector. Sublinear scaling dampens extreme term frequencies.

scikit-learn
03
⚙️

ML Classification

Logistic Regression learns one weight per TF-IDF feature. class_weight='balanced' handles dataset imbalance without resampling.

Logistic Regression
04
📊

Prediction Output

Returns a binary label (0/1) and a calibrated probability score so you always know exactly how confident the model is.

predict_proba
🔬 Model Comparison

Why Logistic Regression?

Three models were benchmarked. LR wins on F1 — the most meaningful metric when missing real disasters is costly.

Model Accuracy Precision Recall F1
⭐ Logistic Regression 80.6% 77.6% 77.2% 77.4%
SVM (LinearSVC) 79.8% 76.5% 76.3% 76.4%
Naive Bayes 81.3% 83.9% 69.9% 76.2%

⚠️ Naive Bayes has the highest precision but the lowest recall — it misses too many real disasters.
